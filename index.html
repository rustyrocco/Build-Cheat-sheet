HTML Headings Reference

Table Tags

Tag Name Description
<h1> Main Heading This is the most important heading, and should ideally appear only once.
<h2> Subheading This is used as a sub heading and can be used as a Topic Heading.
<h3> Lesser Heading This is used as a smaller heading and can be used as an Article Heading.
<h4> Small Heading This is used to typically within a section headed by an <h3>
<h5> Smaller Heading This can be used to be a heading for sub topics.
<h6> Samllest Heading This can also be used to be a heading for sub topics, but is not often used.
